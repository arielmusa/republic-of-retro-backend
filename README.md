# Republic of Retro

Republic of Retro is an e-commerce application where users can browse products,use filters, view details and do cart and checkout actions.

## Project Structure
```
.../
  ├── config/
  ├── controllers/
  ├── middlewares/
  ├── routers/
  ├── public/
  ├── utils/
  └─── app.js
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/arielmusa/republic-of-retro-backend.git
```

Go to the project directory

```bash
  cd republic-of-retro-backend
```

Run the project

```bash
npm install
npm run dev
```

Server will run at:
http://localhost:3000
